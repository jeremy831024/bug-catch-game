# Bug Catch Adventure Godot Prototype

Godot 4.6.x prototype for the 3D sandbox survival version.

## Run

1. Open the `godot/` folder with Godot 4.6.x.
2. Run `scenes/main.tscn`.

Controls:

- `WASD`: move
- `Shift`: sprint
- `E` / `Space`: catch, observe, or interact

The prototype generates a placeholder 3D world at runtime: grassland, lake obstacle, hill, trees, food, hazards and insects.
