# 抓虫大冒险 Unity 沙盒生存美术提示词

## 1. 使用原则

本文件用于生成第一版视觉占位符，以及后续正式模型、贴图、动画的美术方向。提示词可以用于概念图、3D 模型生成、贴图生成、图标生成或给美术团队做风格说明。

统一风格：

- stylized low-poly
- warm summer wilderness
- readable silhouette
- kid-friendly adventure game
- clean shapes
- soft colors
- game-ready asset

通用负面提示词：

```text
no horror, no gore, no realistic insects, no overly detailed texture, no text, no watermark, no dark scary mood, no photorealistic skin, no complex background
```

通用 3D 资产提示词后缀：

```text
low-poly stylized 3D game asset, clean silhouette, separate material zones, Unity-ready, centered, neutral lighting, isolated on plain background, front view and three-quarter view, optimized for real-time game
```

## 8. 生成交付要求

概念图阶段：

- 需要正面、侧面、三分之四视角
- 背景尽量纯色
- 先看 silhouette，不追求细节

3D 模型阶段：

- 模型、贴图、动画分开交付
- 碰撞体不依赖模型网格
- 动画状态名必须和技术设计保持一致
- 低多边形优先，保证 Unity 实时运行
