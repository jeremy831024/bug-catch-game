# 昆虫生态与感知规范

## 1. 设计目标

昆虫不能只是不同分数的移动目标。每种昆虫都应该有自己的生态特性、出没条件、躲藏习性、社交行为和感知反馈。玩家应该能通过声音、大小变化、移动痕迹、环境线索来定位虫子，而不是只能靠屏幕上直接看到虫子。

## 2. 昆虫配置维度

每种昆虫至少需要配置以下字段：

```text
id
displayName
rarity
biomeTags
activeTime
weatherPreference
hideHabit
socialType
movementType
threatResponse
audioCue
visualCue
proximityCue
captureDifficulty
scoreValue
collectionEntry
```

## 3. 生态维度定义

### 3.1 躲藏习性 hideHabit

- `none`：基本不躲藏
- `grass`：藏在草丛
- `tree`：停在树上或树干
- `burrow`：可钻洞
- `rock`：贴近石头、山地边缘
- `swamp`：藏在沼泽湿地
- `web`：依附蜘蛛网或角落
- `camouflage`：通过颜色和环境融合

### 3.2 出没习性 activeTime

- `day`：白天活跃
- `night`：夜晚活跃
- `dusk`：黄昏活跃
- `rain`：雨后或雨中活跃
- `fog`：雾天活跃
- `rareEvent`：随机生态事件触发

### 3.3 社交行为 socialType

- `solo`：单独出现
- `pair`：成对出现
- `smallGroup`：小群体
- `swarm`：群体事件
- `territorial`：有领地，不喜欢其他虫靠近

### 3.4 感知反馈 proximityCue

玩家接近时，昆虫应提供至少一种定位线索：

- 声音变大
- 声音频率变快
- 模型视觉大小或轮廓放大
- 草丛晃动
- 地面粒子
- 飞行轨迹
- 夜间发光
- 树干振动或声波提示

## 4. 声音提示设计

### 4.1 声音距离规则

- 远距离：低音量、低频率、方向不明显
- 中距离：音量上升，左右声道可定位
- 近距离：声音明显，叠加视觉提示

### 4.2 3D 音频要求

- 使用 Unity 3D Audio Source
- 根据距离做音量衰减
- 稀有虫可使用更独特的音色
- 群居虫可使用多声源或环绕声场

### 4.3 声音不应过载

- 同屏多个虫子时，限制同时播放的主动提示音数量
- 高优先级声音：稀有虫、危险虫、任务目标虫
- 普通虫的提示音可降低频率

## 5. 视觉定位设计

### 5.1 大小变化

当玩家接近目标昆虫时，可以轻微增强视觉可读性：

- 轮廓高亮
- 模型缩放 1.05-1.15
- 头顶小提示
- 周围草叶晃动

注意：大小变化不能破坏真实感，应该更像“玩家注意力聚焦”，不是物体突然变大。

### 5.2 环境线索

- 草丛虫：草丛轻微摇晃
- 树栖虫：树干声波或树叶震动
- 洞穴虫：洞口尘土或轻微声音
- 沼泽虫：水面气泡或泥水涟漪
- 夜行虫：微弱发光或反光

## 6. 当前核心虫子设计

### 6.1 蚂蚱

- 躲藏：草丛
- 出没：白天、草地
- 社交：小群体
- 行为：靠近后连续跳跃逃跑
- 声音：短促草叶摩擦声、跳跃落地声
- 视觉提示：草丛晃动、跳跃弧线
- 定位方式：玩家靠近草丛时听到沙沙声，草丛局部抖动

### 6.2 螳螂

- 躲藏：伪装在草叶、灌木、树边
- 出没：白天、草地和林地边缘
- 社交：单独出现
- 行为：静止伪装，玩家靠近才移动或攻击式后撤
- 声音：很轻的肢体摩擦声
- 视觉提示：轮廓在近距离轻微显现
- 定位方式：玩家接近时模型透明度降低，轮廓更明显

### 6.3 独角仙

- 躲藏：树干、朽木、山地边缘
- 出没：黄昏、雨后、林地
- 社交：单独或成对
- 行为：移动慢，受惊后短距离飞行或绕树移动
- 声音：低沉甲壳摩擦声、短促振翅声
- 视觉提示：树干附近有细小木屑或反光
- 定位方式：雨后靠近树木时听到低频振翅声

### 6.4 蝴蝶

- 躲藏：花丛、草地开阔处
- 出没：白天、晴天
- 社交：成对或小群体
- 行为：轻盈飞行，路线飘忽
- 声音：极轻翅膀声，主要靠视觉
- 视觉提示：彩色飞行轨迹、花粉粒子
- 定位方式：远处可看到彩色轨迹，靠近后轨迹更明显

### 6.5 知了

- 躲藏：树干和树冠
- 出没：白天、炎热天气、林地
- 社交：群鸣事件
- 行为：树上停栖，树与树之间飞行
- 声音：标志性鸣叫，距离越近越响
- 视觉提示：树干声波、树叶震动
- 定位方式：玩家靠近某棵树时鸣叫声增强，树干出现短暂声波圈

### 6.6 蜘蛛

- 躲藏：蜘蛛网、洞穴角落、树根
- 出没：夜晚、洞穴、潮湿区域
- 社交：单独或少量
- 行为：布置蜘蛛网，玩家靠近时横向移动或退回网中
- 声音：低频爬行声、网丝拉扯声
- 视觉提示：网丝反光、地面阴影
- 定位方式：手电照到网时出现高亮，靠近有拉丝声

### 6.7 屎壳郎

- 躲藏：草地边缘、泥地、沼泽边
- 出没：白天或雨后
- 社交：单独
- 行为：推球移动，可能挡路或撞到玩家
- 声音：滚动物体声、低频摩擦声
- 视觉提示：泥地滚动痕迹
- 定位方式：玩家靠近时听到滚动声，地面有短暂轨迹

## 8. 开发优先级

### Phase 1

- 给 3 种基础虫加入音效提示和近距离视觉提示
- 实现草丛晃动、树干声波、飞行轨迹三类反馈

### Phase 2

- 所有核心虫子接入生态配置
- 3D 音频距离衰减
- 稀有虫和危险虫提示优先级

### Phase 3

- 群居虫声场
- 天气和昼夜改变声音与出没
- 图鉴中展示习性信息
