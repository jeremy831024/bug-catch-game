# 昆虫种类扩展规划

## 1. 目标

后续增加昆虫时，不只是增加图鉴数量，而是让不同昆虫带来不同的玩法体验。新增物种必须经过三层筛选：

1. 生态合理性：是否适合北京或华北野外环境、草地、林地、湿地、洞穴等场景。
2. 保护合规性：国家重点保护野生动物、地方重点保护或需要谨慎处理的物种，不设计为可捕捉对象。
3. 玩法差异性：新增物种应带来新的声音、行为、躲藏、群居或危险机制。

## 2. 资料依据

规划参考以下资料方向：

- 北京 2020-2024 生物多样性本底调查共记录昆虫 2613 种，说明北京具备足够丰富的本地昆虫池可用于游戏扩展。
- 北京城市绿地和公园蝴蝶调查中记录到菜粉蝶、云粉蝶、黄钩蛱蝶、金凤蝶、柑橘凤蝶等常见或典型物种，可作为北京本地蝴蝶的初始参考。
- 北京蝴蝶记录丰富，公开资料整理北京记录蝴蝶超过 170 种。
- 北京顺义七彩蝶园公开介绍过红翅尖粉蝶、橙粉蝶、红锯蛱蝶、绿带翠凤蝶、琉璃灰蝶、网丝蛱蝶、帅蛱蝶等蝴蝶品类，可作为美术和分类参考，但正式野外物种仍需逐项核验。
- 中国林业科学研究院的科普资料显示铜绿丽金龟分布包含北京，且具有黄昏和夜间活动、趋光等明确玩法特征。
- 国家重点保护野生动物名录是硬性排除规则。
- “三有”野生动物名录是合规与教育提示规则，列入后不宜设计成随意捕捉，需要进一步确认玩法处理方式。
- 胡蜂、马蜂等有蜇刺风险的昆虫可作为危险环境元素，但不作为可捕捉目标。

## 3. 昆虫加入规则

### 3.1 可捕捉

满足以下条件才可加入可捕捉列表：

- 不属于国家重点保护野生动物
- 不属于明确需要禁止捕捉的地方保护物种
- 不属于高风险蜇刺或明显危险物种
- 能提供明确玩法差异

### 3.2 不可捕捉但可观察

适用于：

- 保护物种
- 珍稀物种
- 教育意义较强但不适合捕捉的昆虫

玩法处理：

- 图鉴中可作为“观察记录”
- 玩家靠近只能拍照、记录或获得生态分
- 不允许网捕或陷阱捕获

### 3.3 不可捕捉危险物

适用于：

- 胡蜂、马蜂等蜇刺风险物种
- 强攻击性或有明显安全隐患的昆虫

玩法处理：

- 作为危险区域提示
- 靠近会触发追击、警告音或体力损失
- 不计入捕捉图鉴，可计入生态观察或危险遭遇记录

## 4. 候选可捕捉昆虫

### 4.1 鹿角花金龟

状态：候选，需要进一步核验具体中文名和学名。

玩法定位：

- 稀有甲虫
- 多出现在林地、树液、朽木附近
- 单独出现
- 抓捕价值高

行为建议：

- 白天或黄昏出没
- 靠近时发出低频振翅声
- 受惊后短距离飞行到另一棵树

### 4.2 铜绿丽金龟

状态：北京常见候选，玩法适配度高。

玩法定位：

- 中高频率的林地和果树边缘甲虫
- 黄昏后和夜间更活跃
- 具有明显趋光性，适合手电和营火周边玩法

行为建议：

- 白天隐蔽，傍晚后活动增强
- 对光源有明显反应
- 受惊后快速起飞并绕树移动

### 4.3 白星花金龟 / 东方花金龟类

玩法定位：

- 常见花金龟类
- 可作为早期甲虫扩展
- 适合林地、果实、树液附近

行为建议：

- 飞行声音明显
- 喜食腐果或树液，可被诱饵吸引
- 可设计成“嗡嗡声定位”教学虫

### 4.4 丽金龟类

状态：候选类群，需要选定具体物种，例如铜绿丽金龟等常见类型。

玩法定位：

- 夜晚或灯光附近活动
- 可与手电筒、夜间玩法挂钩

行为建议：

- 夜晚趋光
- 靠近手电光源
- 飞行不稳定，容易撞向光源

### 4.5 蟋蟀类

状态：北京常见夜鸣虫方向，建议优先作为基础声音定位虫。

玩法定位：

- 夜间鸣虫
- 适合声音定位玩法
- 草地、石缝、洞口附近

行为建议：

- 夜晚鸣叫
- 声音越近越清晰
- 玩家靠近后停止鸣叫并躲藏
- 可通过草丛晃动或地面小洞定位

### 4.6 蝗虫 / 蚱蜢类

玩法定位：

- 草地基础虫
- 教学跳跃行为
- 小群体出现

行为建议：

- 靠近后跳跃逃跑
- 群体惊动时连续跳起
- 适合早期抓捕教学

## 5. 蝴蝶扩展方向

蝴蝶不应只做换色，应按照行为、出没时间、飞行方式和稀有度区分。

### 5.1 白色粉蝶类

代表方向：

- 菜粉蝶
- 东方菜粉蝶
- 云粉蝶

玩法：

- 白天常见
- 飞行慢
- 适合新手捕捉

### 5.2 黄色 / 橙色粉蝶类

代表方向：

- 东亚豆粉蝶
- 其他黄色或橙色粉蝶

玩法：

- 花丛和草地出现
- 受天气影响明显
- 可在晴天概率更高

### 5.3 蛱蝶类

代表方向：

- 黄钩蛱蝶
- 红锯蛱蝶
- 网丝蛱蝶
- 帅蛱蝶
- 猫蛱蝶

玩法：

- 飞行路线更快
- 喜欢林缘、花丛、湿地边缘
- 中等难度

### 5.4 凤蝶类

代表方向：

- 金凤蝶
- 柑橘凤蝶

玩法：

- 大型、醒目、高价值
- 飞行速度快
- 适合做稀有非保护目标

### 5.5 灰蝶类

代表方向：

- 蓝灰蝶
- 琉璃灰蝶

玩法：

- 体型小
- 难发现
- 适合近距离视觉提示

## 6. 不加入可捕捉的保护或谨慎物种

以下物种或类群不作为可捕捉目标：

- 金斑喙凤蝶：国家一级保护蝶类
- 金裳凤蝶：国家二级重点保护物种
- 阳彩臂金龟：国家重点保护物种
- 已列入国家重点保护野生动物名录的其他昆虫
- 已列入地方重点保护且不适合捕捉的物种

处理方式：

- 如果出现在游戏中，只作为观察、拍照、生态教育或特殊事件
- 不允许捕捉，不允许陷阱捕获，不进入普通积分体系
- 保护名录不做“换皮替代”，只允许做观察模式或任务事件

## 7. 危险不可捕捉类

### 7.1 胡蜂 / 马蜂

玩法定位：

- 危险环境元素
- 不可捕捉
- 可提示附近有巢穴或高风险区域

行为建议：

- 玩家靠近巢穴时出现警告声
- 持续靠近触发追击
- 被蜇后体力下降、中毒或移动受限
- 解毒药或防护装备可降低伤害

### 7.2 危险蝴蝶 / 昆虫伪装提示

如果后续需要表现“危险但无攻击动作”的昆虫，只做环境警示，不做可捕捉对象：

- 蜂巢附近的胡蜂群
- 毒区周边的高风险飞虫
- 出现时触发红色边界、警示音和体力损失提示

## 9. 分期加入建议

### Phase 1

- 蟋蟀
- 铜绿丽金龟
- 菜粉蝶 / 云粉蝶
- 白星花金龟 / 东方花金龟类

目标：验证声音定位、甲虫飞行声、蝴蝶飞行差异。

### Phase 2

- 丽金龟类
- 黄钩蛱蝶
- 东亚豆粉蝶
- 蓝灰蝶 / 琉璃灰蝶
- 蝗虫 / 蚱蜢细分

目标：丰富夜间、花丛、草地生态。

### Phase 3

- 鹿角花金龟
- 金凤蝶 / 柑橘凤蝶（先观察后决定是否可捕捉）
- 更多稀有非保护蝴蝶
- 胡蜂危险事件

目标：引入高价值目标和危险不可捕捉生态元素。
