# 抓虫大冒险 Unity 沙盒生存 PRD

## 1. 产品定位

抓虫大冒险升级为一款第三人称 3D 沙盒生存抓虫游戏。玩家在随机生成的野外世界中探索、抓捕昆虫、收集图鉴、应对障碍与威胁，并通过补给和道具维持生存。

核心体验不是纯时间竞速，而是“生存压力 + 探索收集 + 资源取舍”。

## 2. 目标用户

- 喜欢轻度生存和收集的玩家
- 喜欢野外探索、昆虫题材、沙盒地图的玩家
- 喜欢第三人称 3D 视角、资源管理和任务推进的玩家

## 3. 平台与形态

- 第一阶段：Unity 3D 单机版
- 后续：可导出 PC/Mac 版本，也可评估 WebGL 版本
- UI 与玩法分离，便于后续迁移到更多平台

## 4. 核心卖点

- 随机地图区块，支持草地、沼泽、洞穴、山地、湖泊
- 第三人称跟随视角，具备物理碰撞
- 昼夜循环和天气系统改变玩法节奏
- 昆虫、障碍、道具和威胁形成多层策略
- 体力生存机制替代计时机制，带来持续压力

## 5. 核心玩法循环

1. 选择预设角色
2. 进入地图
3. 探索地图、寻找虫子与补给
4. 抓捕昆虫、避开威胁、管理体力和背包
5. 使用道具和安全区恢复状态
6. 完成每日任务和长期目标
7. 体力归零则结束游戏

## 6. 角色与成长

### 6.1 预设角色

不做自由创建角色，改为预设角色选择。第一版先用视觉占位符，后续替换模型、贴图和动画。

### 6.2 成长系统

- 体力上限可通过锻炼或道具提升
- 移动速度可升级
- 背包容量有限，需要取舍
- 可携带常规道具和消耗品

## 7. 地图与环境

### 7.1 地图生成

- 随机生成地图区块
- 支持草地、沼泽、洞穴、山地、湖泊
- 地图应具备区域差异和探索层次

### 7.2 昼夜循环

- 白天：视野正常，部分虫子活跃
- 夜晚：视野缩小，某些虫子只在夜间出现
- 手电筒可改善夜间可见度

### 7.3 天气系统

- 下雨：地面更滑，移动惯性增大
- 雾天：视野受限
- 天气变化影响虫子行为和地图风险

## 8. 昆虫系统

### 8.1 昆虫分类

- 逃跑型
- 攻击型
- 伪装型
- 飞行型
- 夜行型
- 稀有型

### 8.2 图鉴系统

- 捕获记录自动写入图鉴
- 显示首次发现时间、累计捕获数、出现区域
- 稀有虫奖励更高

### 8.3 出现规则

- 普通虫：安全区和草地常见
- 稀有虫：危险区域、夜晚、特定天气出现概率更高
- 某些虫只在夜间出现

## 9. 障碍与威胁

- 蜘蛛网：减速陷阱
- 毒蘑菇：触碰后进入中毒状态，持续消耗体力并自动减速
- 捕食者：青蛙、鸟，会主动追击玩家
- 地形陷阱：泥坑、断崖
- 湖泊：不可通行

## 10. 道具系统

### 10.1 消耗品

- 食品：随机生成，恢复体力
- 解毒药：随身携带，减轻中毒伤害

### 10.2 功能道具

- 加速靴：临时提速
- 隐身叶：短暂躲避捕食者
- 陷阱工具：放置后自动捕虫
- 手电筒：夜间照明

### 10.3 背包约束

- 背包容量有限
- 玩家需要决定携带食物、解毒药还是功能道具
- 道具掉落和捡取要有即时反馈

## 11. 生存压力

- 体力持续缓慢消耗，模拟饥饿
- 移动、受伤、中毒、雨天泥地都会加速体力消耗
- 体力归零 = 游戏结束
- 安全区域可休息恢复

## 12. 目标与进度

- 每日任务：捕捉指定昆虫、采集道具、探索区域
- 长期目标：完成图鉴、解锁新区域、达成成就
- 积分排行榜：显示分数、图鉴进度、生存时长或任务完成数

## 13. 视觉与操作要求

- 3D 第三人称视角
- 主角清晰可辨，头顶带跟随泛光灯
- 世界地面保持可读性，草地/棋盘格用于区分区域
- 抓到虫子后必须有明确效果提示
- 物理碰撞要稳定，避免穿模和误判

## 14. MVP 范围

第一阶段必须完成：

- 预设角色选择
- 第三人称 3D 移动
- 随机地图区块
- 草地、湖泊、山地
- 3-5 种虫子
- 食品、解毒药
- 体力系统
- 蜘蛛网或一种减速障碍
- 日夜或天气中的一种
- 图鉴和排行榜

## 15. 暂不做

- 自由捏角色
- 联机对战
- 大型剧情任务
- 复杂战斗系统
- 过早堆叠大量美术资产

## 16. 成功标准

- 玩家能在 3D 地图中稳定移动和抓虫
- 体力压力能驱动探索和资源管理
- 道具、障碍、天气和昼夜会改变玩家决策
- 图鉴、任务和排行榜形成长期目标
