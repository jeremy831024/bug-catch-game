# 装备系统规划

## 1. 目标

装备系统的作用不是单纯加数值，而是帮助玩家在生存压力下做取舍，并形成清晰的玩法身份。

装备要同时服务三件事：

- 提升生存能力
- 改变抓虫效率
- 形成视觉身份和成长感

## 2. 装备边界

- 装备：长期穿戴或佩戴，提供持续属性或被动/主动效果
- 道具：一次性消耗或短时使用
- 背包：临时携带和取舍空间

装备不能和道具混成一类。装备应该有明确槽位、明确属性和明确升级路径。

## 3. 装备槽位

建议槽位如下：

- 头部：帽子、头灯、护目镜
- 上身：衣服、外套、护甲
- 下身：裤子、护腿
- 鞋子：普通鞋、雨靴、加速靴
- 手部：手套、抓握辅助装备
- 背部：背包、呼吸/防护装置
- 工具位：捕虫网、铲子、陷阱工具、手电筒
- 饰品位1：护符、香包、解毒挂件
- 饰品位2：临时增益饰品

## 4. 装备类型

### 4.1 防护类

- 提升中毒抗性
- 降低蜘蛛网减速
- 提升雨天、沼泽、洞穴适应性

### 4.2 机动类

- 提升移动速度
- 降低滑地惩罚
- 提升转向灵活性

### 4.3 采集类

- 提升抓虫判定范围
- 提升挖掘效率
- 减少虫子逃脱概率

### 4.4 续航类

- 提升体力上限
- 降低体力消耗
- 提高休息恢复效率

### 4.5 感知类

- 夜间视野增强
- 昆虫声音提示增强
- 稀有虫定位更明显

## 5. 属性系统

每件装备至少可影响下列属性之一：

- `staminaMax`
- `staminaDrainRate`
- `moveSpeed`
- `turnSpeed`
- `captureRange`
- `captureAccuracy`
- `digSpeed`
- `poisonResistance`
- `slipResistance`
- `nightVision`
- `audioRange`
- `bagCapacity`
- `trapEfficiency`

建议采用“基础值 + 装备修正 + 状态修正”的叠加方式。

## 6. 稀有度

- 普通：基础装备
- 稀有：中等属性提升
- 史诗：带明显玩法变化
- 传奇：有独特被动或主动效果

稀有度不能只加数值，至少要在玩法上有一处明显区别。

## 7. 质量与耐久

### 7.1 耐久

部分装备可有耐久：

- 雨靴在泥地和雨天会损耗
- 手电筒会耗电
- 陷阱工具会消耗次数
- 捕虫网可有磨损

### 7.2 修复

- 可在营地修复
- 可用材料修复
- 高级装备修复成本更高

## 8. 获取方式

- 商店购买
- 任务奖励
- 探索掉落
- 稀有虫事件奖励
- 图鉴/成就解锁

## 9. 升级路径

每件装备可拥有 1-3 个升级段：

- 基础版
- 强化版
- 专精版

升级可改变：

- 数值
- 外观
- 被动效果
- 音效或特效

## 10. 重点装备建议

### 10.1 帽子

- 草帽：默认，轻便
- 雨帽：提升雨天表现
- 头灯帽：提升夜间视野

### 10.2 上衣

- 防护外套：中毒减伤
- 轻便衬衫：提高移动速度
- 口袋背心：增加道具携带上限

### 10.3 鞋子

- 普通鞋：平衡
- 雨靴：沼泽和雨天更稳
- 加速靴：短时提速但耐久高消耗

### 10.4 手部工具

- 标准捕虫网：基础抓捕
- 强化捕虫网：抓捕范围更大
- 轻量捕虫网：挥动更快

### 10.5 背部

- 小背包：默认
- 扩容背包：提高容量
- 防护背包：减伤、抗污

### 10.6 饰品

- 解毒香包：降低中毒时间或伤害
- 护身符：降低捕食者威胁
- 生态徽章：提高某类虫的发现概率

## 11. 与背包的关系

- 装备占固定槽位，不占普通道具背包
- 可携带的备用装备才占背包
- 装备栏和消耗品栏要分开

## 12. UI 与反馈

- 角色身上要能直接看到装备变化
- HUD 需要显示当前装备槽
- 获取装备时有对比提示
- 升级时要显示属性变化

## 13. 资源要求

每件装备后续正式化至少需要：

- 占位符模型
- 正式模型
- 贴图
- 装备动画或附着点
- 装备特效
- 装备音效

## 14. 数据字段建议

```text
id
displayName
slotType
rarity
modelPlaceholder
modelFinal
icon
statModifiers
activeSkill
passiveSkill
durabilityMax
durabilityLossRate
upgradePath
dropSource
sellValue
description
```

## 15. 设计原则

- 装备要让玩家感到“风格不同”
- 装备要和地图生态挂钩
- 装备要有舍弃成本
- 装备要支持后续大量扩展而不推翻系统
