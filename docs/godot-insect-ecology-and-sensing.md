# 昆虫生态与感知规范

## 1. 设计目标

昆虫不能只是不同分数的移动目标。每种昆虫都应该有自己的生态特性、出没条件、躲藏习性、社交行为和感知反馈。玩家应该能通过声音、大小变化、移动痕迹、环境线索来定位虫子，而不是只能靠屏幕上直接看到虫子。

## 2. 昆虫配置维度

每种昆虫至少需要配置以下字段：

```text
id
displayName
rarity
speciesGroup
safetyTag
captureMode
biomeTags
activeTime
weatherPreference
hideHabit
socialType
movementType
threatResponse
audioCue
visualCue
proximityCue
captureDifficulty
scoreValue
collectionEntry
sourceNote
```

## 3. 生态维度定义

### 3.1 躲藏习性 hideHabit

- `none`：基本不躲藏
- `grass`：藏在草丛
- `tree`：停在树上或树干
- `burrow`：可钻洞
- `rock`：贴近石头、山地边缘
- `swamp`：藏在沼泽湿地
- `web`：依附蜘蛛网或角落
- `camouflage`：通过颜色和环境融合

### 3.2 出没习性 activeTime

- `day`：白天活跃
- `night`：夜晚活跃
- `dusk`：黄昏活跃
- `rain`：雨后或雨中活跃
- `fog`：雾天活跃
- `rareEvent`：随机生态事件触发

### 3.3 社交行为 socialType

- `solo`：单独出现
- `pair`：成对出现
- `smallGroup`：小群体
- `swarm`：群体事件
- `territorial`：有领地，不喜欢其他虫靠近

### 3.4 感知反馈 proximityCue

玩家接近时，昆虫应提供至少一种定位线索：

- 声音变大
- 声音频率变快
- 模型视觉大小或轮廓放大
- 草丛晃动
- 地面粒子
- 飞行轨迹
- 夜间发光
- 树干振动或声波提示

### 3.5 物种分组 speciesGroup

用于区分同一类虫的美术和行为模板：

- `beetle`：甲虫类
- `butterfly`：蝴蝶类
- `cricket`：蟋蟀类和鸣虫类
- `grasshopper`：蝗虫 / 蚱蜢类
- `predator`：危险捕食者或危险昆虫

### 3.6 安全标签 safetyTag / captureMode

用于把“能看见”与“能抓”分开：

- `catchable`：可捕捉
- `observe_only`：仅观察，不可捕捉
- `hazard`：危险元素，不可捕捉

captureMode 用于进一步说明交互方式：

- `net_capture`：常规捕虫网捕捉
- `photo_only`：仅拍照记录
- `hazard_event`：危险事件触发，不进入图鉴积分

## 4. 声音提示设计

### 4.1 声音距离规则

- 远距离：低音量、低频率、方向不明显
- 中距离：音量上升，左右声道可定位
- 近距离：声音明显，叠加视觉提示

### 4.2 3D 音频要求

- 使用 Godot `AudioStreamPlayer3D`
- 根据距离做音量衰减
- 稀有虫可使用更独特的音色
- 群居虫可使用多声源或环绕声场

### 4.3 声音不应过载

- 同屏多个虫子时，限制同时播放的主动提示音数量
- 高优先级声音：稀有虫、危险虫、任务目标虫
- 普通虫的提示音可降低频率

## 5. 视觉定位设计

### 5.1 大小变化

当玩家接近目标昆虫时，可以轻微增强视觉可读性：

- 轮廓高亮
- 模型缩放 1.05-1.15
- 头顶小提示
- 周围草叶晃动

注意：大小变化不能破坏真实感，应该更像“玩家注意力聚焦”，不是物体突然变大。

### 5.2 环境线索

- 草丛虫：草丛轻微摇晃
- 树栖虫：树干声波或树叶震动
- 洞穴虫：洞口尘土或轻微声音
- 沼泽虫：水面气泡或泥水涟漪
- 夜行虫：微弱发光或反光

## 6. 当前核心虫子设计

### 6.1 蚂蚱

- 躲藏：草丛
- 出没：白天、草地
- 社交：小群体
- 行为：靠近后连续跳跃逃跑
- 声音：短促草叶摩擦声、跳跃落地声
- 视觉提示：草丛晃动、跳跃弧线
- 定位方式：玩家靠近草丛时听到沙沙声，草丛局部抖动

### 6.2 螳螂

- 躲藏：伪装在草叶、灌木、树边
- 出没：白天、草地和林地边缘
- 社交：单独出现
- 行为：静止伪装，玩家靠近才移动或攻击式后撤
- 声音：很轻的肢体摩擦声
- 视觉提示：轮廓在近距离轻微显现
- 定位方式：玩家接近时模型透明度降低，轮廓更明显

### 6.3 独角仙

- 躲藏：树干、朽木、山地边缘
- 出没：黄昏、雨后、林地
- 社交：单独或成对
- 行为：移动慢，受惊后短距离飞行或绕树移动
- 声音：低沉甲壳摩擦声、短促振翅声
- 视觉提示：树干附近有细小木屑或反光
- 定位方式：雨后靠近树木时听到低频振翅声

### 6.4 蝴蝶

- 躲藏：花丛、草地开阔处
- 出没：白天、晴天
- 社交：成对或小群体
- 行为：轻盈飞行，路线飘忽
- 声音：极轻翅膀声，主要靠视觉
- 视觉提示：彩色飞行轨迹、花粉粒子
- 定位方式：远处可看到彩色轨迹，靠近后轨迹更明显

### 6.5 知了

- 躲藏：树干和树冠
- 出没：白天、炎热天气、林地
- 社交：群鸣事件
- 行为：树上停栖，树与树之间飞行
- 声音：标志性鸣叫，距离越近越响
- 视觉提示：树干声波、树叶震动
- 定位方式：玩家靠近某棵树时鸣叫声增强，树干出现短暂声波圈

### 6.6 蜘蛛

- 躲藏：蜘蛛网、洞穴角落、树根
- 出没：夜晚、洞穴、潮湿区域
- 社交：单独或少量
- 行为：布置蜘蛛网，玩家靠近时横向移动或退回网中
- 声音：低频爬行声、网丝拉扯声
- 视觉提示：网丝反光、地面阴影
- 定位方式：手电照到网时出现高亮，靠近有拉丝声

### 6.7 屎壳郎

- 躲藏：草地边缘、泥地、沼泽边
- 出没：白天或雨后
- 社交：单独
- 行为：推球移动，可能挡路或撞到玩家
- 声音：滚动物体声、低频摩擦声
- 视觉提示：泥地滚动痕迹
- 定位方式：玩家靠近时听到滚动声，地面有短暂轨迹

### 6.8 铜绿丽金龟

- 躲藏：树叶、果树、灯光附近
- 出没：黄昏、夜晚、晴热天气
- 社交：单独或少量
- 行为：趋光、短距离乱飞、停到树叶或枝头
- 声音：明显嗡嗡振翅声，靠近后更响
- 视觉提示：光源附近反光、树叶轻抖
- 定位方式：玩家携带手电或接近营火时，虫子向光源聚集

### 6.9 鹿角花金龟

- 躲藏：树液点、朽木、树干裂缝
- 出没：黄昏、夏季、林地
- 社交：单独
- 行为：短距离飞行，受惊后飞向另一棵树
- 声音：低频大翅振动声
- 视觉提示：树干附近出现缓慢移动的大型甲虫轮廓
- 定位方式：靠近树液点时，先听到低频振翅，再看到树干反光

### 6.10 蟋蟀

- 躲藏：草丛、石缝、洞口、倒木边缘
- 出没：夜晚、傍晚、温暖天气
- 社交：单独或小群体
- 行为：鸣叫、受惊后钻草或钻洞
- 声音：连续鸣叫，越近越清晰，适合左右声道定位
- 视觉提示：草叶轻颤、洞口附近短促跳动
- 定位方式：玩家先听声再找草丛、洞口和石缝

### 6.11 蝴蝶扩展组

#### 6.11.1 菜粉蝶 / 云粉蝶

- 躲藏：花丛、开阔草地
- 出没：白天、晴天
- 社交：成对或小群体
- 行为：慢速飘飞、停花吸食
- 声音：极轻翅声，主要靠视觉和花粉粒子
- 视觉提示：白色或浅色飞行轨迹
- 定位方式：远处优先看轨迹，近处看花间停落点

#### 6.11.2 黄钩蛱蝶 / 红锯蛱蝶 / 网丝蛱蝶 / 帅蛱蝶

- 躲藏：林缘、湿地边、花丛
- 出没：白天、风小天气
- 社交：单独或小群
- 行为：飞行更快，路线弯折明显
- 声音：轻微扑翅声
- 视觉提示：橙色、红色或黑纹翅面更醒目
- 定位方式：靠近后以飞行方向和颜色对比判断

#### 6.11.3 金凤蝶 / 柑橘凤蝶

- 躲藏：花丛、树缘、温暖空地
- 出没：白天、晴天、暖风天气
- 社交：单独
- 行为：大型、快速、滑翔感强
- 声音：轻微但可辨识的翅声
- 视觉提示：大翅展和高对比纹样
- 定位方式：远距离就能看见大体型轮廓

#### 6.11.4 蓝灰蝶 / 琉璃灰蝶

- 躲藏：低矮草丛、花叶背面
- 出没：白天
- 社交：单独或少量
- 行为：小型、停落快、受惊迅速上窜
- 声音：几乎无明显声音
- 视觉提示：蓝灰色小面积反光
- 定位方式：依赖草丛晃动和近距离轮廓变化

### 6.12 胡蜂 / 马蜂

- 躲藏：蜂巢、树洞、屋檐边缘、灌木深处
- 出没：白天、闷热天气
- 社交：群体
- 行为：巡逻、防御、追击
- 声音：高频蜂鸣和持续警示声
- 视觉提示：蜂群往返、巢穴高亮、危险边界
- 定位方式：靠近巢穴先触发警告，再触发逃离或受伤

## 8. 开发优先级

### Phase 1

- 给 3 种基础虫加入音效提示和近距离视觉提示
- 实现草丛晃动、树干声波、飞行轨迹三类反馈

### Phase 2

- 所有核心虫子接入生态配置
- 3D 音频距离衰减
- 稀有虫和危险虫提示优先级

### Phase 3

- 群居虫声场
- 天气和昼夜改变声音与出没
- 图鉴中展示习性信息
