# 🐛 抓虫大冒险 · 完整游戏设计文档

## 1. 游戏概述

> 一款适合6-12岁儿童的抓虫冒险游戏
> 在野外探索、追踪昆虫、策略抓捕

- **平台**: Web（键盘操作）
- **视角**: 2D 俯视图
- **画风**: 清新卡通，色彩鲜艳
- **受众**: 6-12岁儿童
- **AI素材**: 使用 gpt-image-2 生成昆虫图片和动态效果

## 3. 角色系统

### 小朋友（玩家）

```
🧑 小朋友
├── 属性
│   ├── 体力 💪     MAX: 100
│   ├── 速度 🏃     = 基础速度 × 地形系数 × (体力/100)
│   ├── 状态 🟢     🫨正常 / 🟡中毒
│   └── 装备
│       ├── 抄子 🪟   → 抓虫用（按 E / 空格）
│       └── 铲子 🔧   → 挖洞用（按 Q）
│
├── 背包 📦
│   └── 已捕获昆虫记录
│       ├── 蚂蚱 × 3
│       ├── 螳螂 × 1
│       └── ...
│
└── 分数 ⭐ → 当前总积分
```

### 操作方式

| 按键 | 功能 |
|------|------|
| WASD / 方向键 | 移动小朋友 |
| E / 空格 | 使用抄子抓虫（前方一格范围） |
| Q | 使用铲子挖洞（站在洞口时） |
| 鼠标点击 | 辅助操作（可选） |

## 4. 昆虫系统

### 昆虫图鉴

| 昆虫 | 稀有度 | 速度 | 飞行 | 分值 | 毒 | 洞类型 | 说明 |
|------|--------|------|------|------|-----|--------|------|
| 🦗 蚂蚱 | 普通 | ⭐⭐ | 短距 | 10分 | ❌ | 土洞 | 会跳，跳跃移动 |
| 🦟 螳螂 | 普通 | ⭐⭐ | 短距 | 15分 | ❌ | 草洞 | 会埋伏，不动时难发现 |
| 🪲 独角仙 | 稀有 | ⭐ | 可飞 | 50分 | ❌ | 树洞 | 大只，飞得慢但耐力好 |
| 🦋 蝴蝶 | 稀有 | ⭐⭐⭐ | 可飞 | 30分 | ❌ | 花洞 | 飞行轨迹飘忽 |
| 🦗 知了 | 普通 | ⭐ | 可飞 | 20分 | ❌ | 树洞 | 停在树上，发出声音提示 |
| 🕷️ 蜘蛛 | 少见 | ⭐⭐ | 不能飞 | 25分 | ⚠️ 毒 | 网洞 | 有毒，碰触会中毒 |
| 💩 屎壳郎 | 陷阱 | ⭐ | 不能飞 | -10分 | ⚠️ 毒 | 粪洞 | 扣分陷阱，碰触中毒 |

### 昆虫 AI

```
每帧更新:
├── 移动
│   ├── 地面移动: 随机方向漫步
│   ├── 飞行: 消耗体力 (每秒-2), 速度+50%
│   │   └── 体力=0 → 降落, 自然回复 (每秒+1)
│   └── 地形影响: 草地正常, 山路减速
│
├── 进洞行为
│   ├── 靠近洞口 → 概率进入 (体力>30%)
│   ├── 洞内倒计时 (5~15秒)
│   │   └── 超时 → 自动出洞, 体力-10%
│   └── 出洞后冷却才可再进
│
└── 受惊反应
    └── 玩家靠近 (3格内) → 加速逃跑
```

| 🦗 知了 | 🫨 树干洞 (连通型) | 多个出口 |
| 🕷️ 蜘蛛 | 🕸️ 蛛网洞 (死路型) | 单出口 |
| 💩 屎壳郎 | 💩 粪球洞 (死路型) | 单出口 |

## 5. 铲子系统

| 场景 | 结果 |
|------|------|
| 挖 **死路型洞口** + **有虫** | ✅ 额外 +30分 + 虫子奖励 + 少量体力消耗(-10) |
| 挖 **死路型洞口** + **无虫** | ❌ 体力消耗较大 (-25) |
| 挖 **连通型洞口** | ❌ 破坏洞穴, 体力消耗 (-15) |

## 6. 中毒系统

| 触发方式 | 效果 | 解除 |
|----------|------|------|
| 抓毒虫 (蜘蛛/屎壳郎) | 🟡 中毒状态 | 等待 8秒自动解毒 |
| 挖有蜘蛛的洞 | 🟡 中毒状态 | 等待 8秒自动解毒 |
| 中毒期间 | 不能移动 | 每秒减 5分 |
| 解除后 | 恢复正常 | - |

## 7. 体力系统

```
体力 ──────── 100 (满)
    ├── 移动 → 不变 (自然恢复)
    ├── 跑步 → 每秒 -1
    ├── 飞行捕虫 → 一次 -5
    ├── 铲子挖洞 → 一次 -10~25
    ├── 中毒 → 不能恢复
    └── 自然恢复 → 每秒 +1 (最大 100)
    
体力影响:
    体力 > 70 → 速度 100%
    体力 40~70 → 速度 80%
    体力 < 40 → 速度 60%
    体力 = 0 → 不能跑，只能走
```

## 10. 技术方案

### 架构

```
index.html       ← 入口页面
├── style.css    ← 样式 (毛玻璃UI + 游戏画布样式)
├── game.js      ← 主游戏引擎
│   ├── Map       ← 地图/地形系统
│   ├── Player    ← 玩家角色
│   ├── Bug       ← 昆虫AI系统
│   ├── Holes     ← 洞穴系统
│   ├── Shovel    ← 铲子系统
│   ├── Poison    ← 中毒系统
│   ├── Stamina   ← 体力系统
│   └── UI        ← HUD渲染
├── assets/       ← 生成的图片资源
│   ├── player.png
│   ├── bug-grasshopper.png
│   ├── bug-mantis.png
│   ├── bug-beetle.png
│   ├── bug-butterfly.png
│   ├── bug-cicada.png
│   ├── bug-spider.png
│   ├── bug-dung.png
│   └── tiles/   ← 地图瓦片
└── prompts/
    └── DESIGN.md
```

### 渲染

- **Canvas 2D** 主游戏渲染
- **素材**: 使用 gpt-image-2 生成昆虫精灵图
- **帧率**: 60fps 固定时间步长

### AI素材生成

使用 gpt-image-2 (通过中转站 API) 生成:
- 各昆虫32x32像素精灵图 (带透明背景)
- 地图瓦片 (草地/山路/池塘)
- UI图标 (抄子/铲子/体力)
- 小朋友角色精灵
