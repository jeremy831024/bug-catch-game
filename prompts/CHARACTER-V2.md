# 角色自定义系统 v2 - 产品方案

## 核心公式

> **组合数 = 款式 × 款式 × 款式 × 款式 × 款式 × 颜色**

## 选项矩阵

| 部位 | 款式 (5种) | 颜色 (每款6-12色) | 组合 |
|------|-----------|-------------------|------|
| 🧢 **帽子** | 棒球帽 / 草帽 / 探险帽 / 毛线帽 / 遮阳帽 | 每款配2-3种主色 | 5×3=15 |
| 👦 **发型+脸** | 短发男孩 / 长发女孩 / 双辫子 / 卷发 / 刺头 | 金/棕/黑/红/灰 | 5×5=25 |
| 👕 **上衣** | T恤 / 背心 / 连帽衫 / 衬衫 / 马甲 | 12色 | 5×12=60 |
| 👖 **下装** | 短裤 / 长裤 / 裙子 / 背带裤 / 工装裤 | 8色 | 5×8=40 |
| 👟 **鞋子** | 运动鞋 / 靴子 / 凉鞋 / 雨鞋 / 帆布鞋 | 6色 | 5×6=30 |

**总组合数: 15 × 25 × 60 × 40 × 30 = 27,000,000 种不同角色！**

## 技术实现

用 Canvas 在选人界面逐层绘制:

```
1. 鞋子 (底层)
2. 裤子/裙子
3. 上衣
4. 手臂
5. 头+脸 (肤色 + 表情)
6. 头发
7. 帽子 (顶层)
8. 抄子装饰 (道具)
```

每层根据用户选择的款式+颜色参数绘制, 实时刷新。
